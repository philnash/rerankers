🔀 rerankers

Use open reranking models to compare documents against a query in the browser or Node.js. This demo runs entirely in your browser; the first run downloads the model and caches it locally.

Use it in your project

Install the package, then import Reranker to rank documents locally in your JavaScript or TypeScript application.

Install with npm npm install rerankers
